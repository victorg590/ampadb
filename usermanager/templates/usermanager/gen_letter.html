Generar circular

Per a crear una circular, escriu una plantilla. Pots utilitzar Markdown. Les següents variables seran substituïdes per a cada alumne:

  • $nom: Nom de l'alumne.
  • $cognoms: Cognoms de l'alumne.
  • $classe: La classe actual de l'alumne.
  • $usuari: El nom d'usuari.
  • $codi: El codi per a activar $usuari.
  • $url: Direcció d'aquest lloc web.
  • Qualsevol altre variable començant per $ és incorrecte i no es modificarà. Ex.: $abcd → "$abcd"
